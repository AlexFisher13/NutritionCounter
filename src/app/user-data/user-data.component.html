Рекомендации

По каким то там стандартам в среднем каждый килограмм нашего тела ежедневного нуждается в 30 ккал. Соответственно для нормального сущестования Вам необходимо потреблять {{ration}} ккал.
Для эффективного жиросжигания следует пребывать в небольшом дефиците каллорий, который не должен превышать 10% основного рациона. В противном случае организм испытывает стресс, что в итоге ведет ко многим неприятным последствиям.
Кол-во белка не должно быть ниже 1,5г на кг веса, и для женщин 1,2, так как белок основной строительный элемент для мышц. Для людей с повышенной физ. активностью долю белка стоит повышать на 15-30%. Долей жиров не стоит принебрегать, так как жиры важнейший нутриент, большое кол-во гормонов синтезируются из жиров и тд. Долей углеводов стоит жервовать во время диеты.
